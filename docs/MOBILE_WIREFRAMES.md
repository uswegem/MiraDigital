## ğŸ” Security Features

1. **PIN Authentication** - 4-6 digit PIN with lockout after 5 failed attempts
2. **Biometric Auth** - Fingerprint/Face ID support
3. **Session Timeout** - Auto-logout after 5 minutes of inactivity
4. **Secure Storage** - Keychain (iOS) / Keystore (Android) for tokens
5. **Certificate Pinning** - Prevent MITM attacks
6. **Device Binding** - Optional device registration
7. **Transaction OTP** - SMS verification for transfers above threshold
8. **Screenshot Prevention** - Disable screenshots on sensitive screens
